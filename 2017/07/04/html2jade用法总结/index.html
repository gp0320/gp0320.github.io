html2jade用法总结

2017-07-04

安装

参数说明

  • -d, –double - 为属性使用双引号
  • -o, –outdir - 输出生成的文件路径
  • -n, –nspaces - 缩进生成文件的空格数。默认为2个空格
  • –donotencode - 中文编码
  • –bodyless - 不包含 html + body 标签
  • –noattrcomma - 属性之间不要逗号分隔
  • –noemptypipe - omit lines with only pipe (‘|’) printable character 去掉|

中文编码

当我们使用 html2jade 转换含有中文的html时，若是在终端中转换只用加个参数 --donotencode 就好了。
